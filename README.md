# 🌊 Flood Early Warning System
**Machine Learning-Powered Flood Prediction | Flask Web Application**

---

## Project Overview
This system predicts flood risk using historical meteorological data. It trains four ML classifiers — Decision Tree, Random Forest, KNN, and XGBoost — and deploys the best-performing model (XGBoost, ~96.55% accuracy) via a Flask web application.

---

## Project Structure
```
flood_project/
│
├── flood_data.csv                # Historical rainfall dataset (2500 records)
├── flood_prediction.ipynb        # Complete ML pipeline (Epics 1–4)
│
├── app.py                        # Flask web application (Epic 5)
├── flood_xgb_model.pkl           # Saved XGBoost model  ← generated by notebook

│
├── templates/
│   └── index.html                # Web UI
│
├── requirements.txt              # Python dependencies
└── README.md                     # This file
```

## Quick Start

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Train the model (run notebook first)
Open `flood_prediction.ipynb` in Jupyter and run all cells. This generates:
- `flood_xgb_model.pkl`
- `scaler.pkl`

```bash
jupyter notebook flood_prediction.ipynb
```

### Step 3 — Run the Flask app
```bash
python app.py
```
Open your browser at: **http://localhost:5000**

## API Reference

### `POST /predict`
**Request body (JSON):**
```json
{
  "ANNUAL":  1250.0,
  "Jan_Feb":   50.0,
  "Mar_May":  120.0,
  "Jun_Sep":  900.0,
  "Oct_Dec":  140.0,
  "YEAR":    2024
}
```

**Response:**
```json
{
  "prediction":   1,
  "label":        "🌊 FLOOD RISK DETECTED",
  "risk_level":   "HIGH",
  "color":        "danger",
  "confidence":   "97.45%",
  "flood_prob":   "94.20%",
  "advice":       "Immediate action required. Issue evacuation advisories..."
}
```

## Model Performance

| Model         | Accuracy | ROC-AUC |
|---------------|----------|---------|
| Decision Tree | ~90%     | ~0.90   |
| Random Forest | ~94%     | ~0.97   |
| KNN           | ~92%     | ~0.95   |
| **XGBoost**   | **~96.55%** | **~0.99** |

---

## Features Used
| Feature   | Description                  |
|-----------|------------------------------|
| ANNUAL    | Total annual rainfall (mm)   |
| Jan-Feb   | Winter season rainfall (mm)  |
| Mar-May   | Pre-monsoon rainfall (mm)    |
| Jun-Sep   | Monsoon season rainfall (mm) |
| Oct-Dec   | Post-monsoon rainfall (mm)   |
| YEAR      | Observation year             |
